# Shamoji-Portfolio

## 概要

ruby on railsで作成したポートフォリオ用のWebアプリ開発作品


## devel環境構築

以下のコマンドを実行してください。

```
bundle install --path vendor/bundle
```

## ローカルサーバ実行

```
bundle exec rails server
```

## Ruby version

* Ruby version : 2.5.8
